Projects

Mobile Barber Application

A mock Airbnb application that finds all available house call barbers in the area. Users can browse through barbers in their area and book appointments with available services. Made with React, TypeScript, CSS, and Material UI for the frontend. Uses Node.js, Express, and a Google Maps API for the backend.

Hexapong

A video game created with Unity and C# inspired by pong and brick breaker. The goal is to prevent the ball from hitting the hexagon by using the shield. You can rotate by using the left and right arrows. Currently working on the graphics and making new levels

Android E-commerce App

An android app built with Java and Firebase that allows customers to browse through stores and services and place orders. It also allows business owners to post their store and their services/inventory. Worked in an Agile environment with 4 other students.

Star Wars Random Character Generator

A React project that generates a random start wars character with their height and homeworld using a Django REST API called SWAPI.

The Walking Dead Simulation

A text-based adventure game based on the Walking Dead made with Java and the use of Object Oriented Programming concepts.
